# Loan Approval Prediction using Decision Tree (Gini Index)

This project implements a Decision Tree classifier using the Gini Index
to predict loan approval outcomes.

## Dataset
Realistic Loan Approval Dataset (US & Canada)

## Model
- Decision Tree Classifier
- Criterion: Gini Index

## Metrics
- Accuracy
- Confusion Matrix
- Classification Report

## How to Run
```bash
pip install -r requirements.txt
python decision_tree_gini.py
